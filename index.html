Projeto atual

API IRIS - Python e Machine Learning

Essa API, foi elaborada para saber qual tipo de flor pertence os dados: Iris-Setosa, Iris-Versicolor ou Iris-Virginica. A API apresenta 4 inputs de entrada no qual o usuario digita:
Sepal Length (Comprimento da sépala) Sepal Width (Largura da sépala)
Petal Length (Comprimento da pétala) Petal Width (Largura da pétala)
Após insertir os dados na API clicando em verificar temos uma segunda tela, que vem da consulta da API trazendo o resultado de qual flor é os inputs digitados. Foi usado neste projeto o front-end e back-end. A inteligencia artificial usada é a Logistic Regression feita em python. As linguagens usadas são flask com machine learning, Python e skylearning.

Tecnologias deste portifolio:

  • HTML
  • CSS
  • SVG
  • PYTHON
  • FLASK
  • SCIKIT-LEARN - Logistic regression
